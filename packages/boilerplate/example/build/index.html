Component Boilerplate

Example

Click a box to trigger the eventHandler

Installation

$ npm i -S @stormid/boilerplate

Usage

HTML

<div class="js-boilerplate"></div>
<div class="js-boilerplate"></div>

JS

Either import and initialise using a named import:

import Boilerplate from '@stormid/boilerplate';
Boilerplate.init('.js-boilerplate');

Or load the UMD version and initialise using the global module name:

import Load from '@stormid/load';
await Load('https://unpkg.com/stormid/umd/@stormid/boilerplate.js');
Boilerplate.init('.js-boilerplate');
